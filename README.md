# FocusFlow

FocusFlow is your calm focus companion designed to help you build consistency through short, achievable focus sessions. Start small, stay focused, grow daily.

### 🌐 Live Deployment
**[https://focusshub.netlify.app](https://focusshub.netlify.app)**

## Features
- **Focus Timer**: Dedicated timer for distraction-free sessions.
- **Dashboard & Stats**: Visualize your focus habits and maintain your streak.
- **Ambient Visuals**: Clean, dynamic aesthetic for a peaceful working environment.
- **Data Privacy**: 100% offline-first. All data is saved directly in your browser's LocalStorage.

## Running Locally
Simply open `index.html` in your browser or serve the directory using any local development server.
